# Threat Model

This lab is educational.

## In scope

- local UDP protocol behavior
- controlled injection hypothesis
- runtime event collection
- evidence packaging
- reproducibility documentation

## Out of scope

- attacking third-party systems
- real-world exploitation
- credential collection
- destructive payloads
- stealth, persistence or evasion
- bypass, persistence or operational offensive tooling

## Safety rule

The lab must remain local, controlled and reproducible.
