# Icare Sentinel Lab — Lab Intent

Icare Sentinel Lab is the advanced continuation of the BEUI2 network/cybersecurity exercise.

It remains educational, but it is now structured as a Doctrine Runtime Evidence Provider.

## Original learning goals

- C
- Linux
- UDP sockets
- protocol reasoning
- tcpdump / network observation
- local injection reasoning
- cybersecurity analysis

## Extended learning goals

- strict build expectations
- reproducible audit outputs
- runtime evidence generation
- basic findings
- SHA-256 manifests
- Doctrine-compatible evidence pack export
- clean separation between lab evidence and Doctrine Platform premium processing

## Product boundary

This public repository does not provide the complete Doctrine Platform.

It only produces runtime evidence that Doctrine Platform can later ingest, report, index, package, seal and sign.
