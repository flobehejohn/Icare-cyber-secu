# Doctrine Compatibility

Icare Sentinel Lab is a public Doctrine Runtime Evidence Provider.

## Provider type

Runtime Network Evidence Provider.

## Domain

- UDP protocol behavior
- runtime network observation
- controlled injection hypothesis
- naive versus hardened protocol reasoning
- educational cybersecurity lab

## Public outputs

The lab can produce:

- evidence-pack.json
- runtime-events.json
- runtime-events.ndjson
- findings.json
- control-mapping.json
- report.md
- report.html
- SHA256SUMS
- summary.json

## Premium boundary

This repository does not include:

- official Doctrine certification
- enterprise scoring
- multi-lab governance dashboard
- keyless signing
- WORM archive enforcement
- commercial seal

Those remain Doctrine Platform responsibilities.
